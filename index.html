Quadro Comparativo das Linguagens

Linguagem Ano Tipagem Criador Descrição
Javascript 1995 Dinâmica e fraca Brendan Eich Conhecida como uma das três principais tecnologias do desenvolvimento web, JavaScript é uma linguagem muito valorizada nos dias atuais e extremamente utilizada por desenvolvedores Front-End.
Java 1995 Estática James Gosling; Patrick Naughton e Sun Fellow Java é uma linguagem de programação destacada por ser orientada a objetos e possuir alto nível de segurança em relação às outras linguagens. Quando surgiu, a ideia principal do Java era que os aparelhos pudessem se comunicar entre si. É reconhecida por possuir portabilidade para qualquer ambiente e desenvolver para múltiplas plataformas.
PHP 1994 Dinâmica Rasmus Lerdorf PHP é uma linguagem usada para desenvolvimento de aplicações atuantes do lado do servidor. Ela é muito utilizada para gerar conteúdos dinâmicos em sites. Por ser muito simples, muitas pessoas iniciantes no universo de programação optam por aprender a linguagem, pois apesar da simplicidade você pode fazer muitas coisas com PHP.
Python 1991 Dinâmica e forte Guido van Rossum Python é uma linguagem muito útil para desenvolver para o lado do servidor. Por possuir uma sintaxe mais simples, é uma das linguagens mais escolhidas por aqueles que estão começando. Ela também é destaque na área de ciência de dados e possui uma vasta variedade de bibliotecas para desenvolvimento de IA.
C# 2000 Forte Microsoft C# é uma linguagem orientada a objetos, multiplataforma, criada pela Big Tech Microsoft. Ela possui sua sintaxe baseada no C++ porém possui influências de outras linguagens como Java.
Kotlin 2011 Estática JetBrains Kotlin é uma linguagem multiplataforma bastante utilizada no desenvolvimento mobile. Ela possui uma sintaxe bastante parecida com C#, Java e Scala, porém é considerada um pouco mais fácil de ler. Se tornou uma linguagem oficial para desenvolvimento de aplicativos android e é utilizada por diversas empresas como Uber, Netflix, Trello, Amazon e Google.
